# PASpipline

### A pipeline to quantify polyA site (PAS) counts from single-cell RNA-Seq R2.

## Installation
```bash
cd envs
conda env create -f env_APA.yml
conda env create -f env_scraps_conda.yml
conda env create -f env_scTail.yml
```

## Run
```bash
bash main.sh
```
